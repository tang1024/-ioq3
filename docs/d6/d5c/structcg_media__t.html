Detailed Description

All of the model, shader, and sound references that are loaded at gamestate time are stored in cgMedia_t.

Other media that can be tied to clients, weapons, or items are stored in the clientInfo_t, itemInfo_t, weaponInfo_t, and powerupInfo_t.

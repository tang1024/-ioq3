186 /*
187  * Here are the definitions for Ravensoft's model format of md4. Raven stores their
188  * playermodels in .mdr files, in some games, which are pretty much like the md4
189  * format implemented by ID soft. It seems like ID's original md4 stuff is not used at all.
190  * MDR is being used in EliteForce, JediKnight2 and Soldiers of Fortune2 (I think).
191  * So this comes in handy for anyone who wants to make it possible to load player
192  * models from these games.
193  * This format has bone tags, which is similar to the thing you have in md3 I suppose.
194  * Raven has released their version of md3view under GPL enabling me to add support
195  * to this codebase. Thanks to Steven Howes aka Skinner for helping with example
196  * source code.
197  *
198  * - Thilo Schulz (arny@ats.s.bawue.de)
199  */
200
